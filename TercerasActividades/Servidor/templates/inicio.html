- Manual de usuario -

Inicio

En este apartado se ofrece una breve explicación sobre el contenido de cada sección de la página web, así como una guía sobre los pasos a seguir para interactuar con la aplicación. Además, se incluyen tablas que muestran las características comunes de los elementos pertenecientes a un mismo grupo, tanto para el parámetro de categoría de ocupación como para el parámetro de nivel educativo.


Aplicación

En este apartado se describe la funcionalidad de la aplicación. El usuario deberá seleccionar uno de los dos parámetros disponibles: "Nivel educativo" o "Categoría de empleo". Luego, deberá hacer clic sobre una entidad federativa en el mapa interactivo. A partir de estas acciones, se desplegarán diversos gráficos que permitirán al usuario visualizar la información.
El usuario podrá consultar la información con el parámetro y la entidad federativa que desee, las veces que así lo requiera.
