Inicio

En este apartado se encuentra una breve explicacion sobre que se muestra en cada apartado asi como los pasos a realizar para interactuar con la aplicación.


Aplicación

En este apartado se encuentra la funcionalidad de la aplicacion. El usuario debera seleccionar uno de los tres siguientes valores como parametro: 'Tipo de cancer', 'Nivel educacional' y 'Categoría de empleo' posteriormente dar click sobre una entidad federativa en el mapa interactivo y visualizar la informacion que se genera en la grafica de barras.
El usuario podra consultar la informacion con el parametro y entidad federativa que desee las veces que asi lo requiera.


Conjunto de datos

En este apartado el usuario podra encontrar cada uno de los conjuntos de Datos que se procesaron y limpiaron en el proyecto asi como el conjunto de datos original (sin procesar) y el conjunto de datos final para que el usuario pueda descargarlo y utilizarlo en sus propios proyectos si asi lo desea.
